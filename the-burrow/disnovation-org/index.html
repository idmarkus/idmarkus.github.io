BASED ON STATE OF THE ART BIG DATA ANALYTICS TECHNIQUES, THIS WORK GENERATES A SERIES OF HIGHLY DETAILED DIGITAL PROFILES OF BIG TECH COMPANIES — IE. PSYCHOLOGICAL, CULTURAL AND POLITICAL PROFILES — SIMILAR TO THE ONES CONSTANTLY GENERATED FOR EACH USER BY THESE VERY SAME COMPANIES. IN OTHER WORDS, THIS ALGORITHM PERFORMS A SOUSVEILLANCE OF BIG TECH COMPANIES USING THEIR OWN TOOLS.
